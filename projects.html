Projects

Mixtures of Tree Bayesian Networks using EM (2021): Implemented the EM algorithm for Mixtures of Tree Bayesian Networks to learn the structure and parameters of the model.

Image Compression using K-means clustering (2021): Built an image compression algorithm using the K-means clustering technique and achieved a significant compression ratio.

Collaborative Filtering (2021): Implemented the Collaborative Filtering algorithm to predict ratings in the test set of Netflix Prize using ratings provided in the training set.

Score-keeper (2021): An end to end system where screenshot of a match leaderboard from Dream11 (fantasy sports platform) is added to an Android app which extracts data from and uploads it to a Google Apps Script web app which processes the data and displays various stats and charts in Google Sheets.

Elegant Popover (2020): A CocoaPod that facilitates highly customisable popovers with multiple borders, borders styles (color and gradient) and arrow styles in iOS. The framework takes care of all the geometrical implementation details within its own subclass of UIPopoverBackgroundView.

FanBrawl API (2020): A Node.js REST API using Express.js and MongoDB that serves as a scorekeeper for users competing on fantasy sports platforms. Users can create groups and compete with each other over the course of different tournaments.

Fitbit Clock-faces (2020): Designed and built several Fitbit clock-faces having features including stats, weather, themes, settable texts and configurable sweeping seconds hand.

Clash of Memories (2019): A two-player memory game in iOS (Swift) using Firebase as server, developed over the course of a 30 hour intra-company hackathon organized by Games24x7. The game was about two players competing in realtime to open all the tiles while also getting penalised when the opponent advances. It won us the top prize.

Word War (2018): A two-player game in Java using Java Swing and Sockets in which the players compete to find more number of words in a predefined time limit from an identical 4x4 board of letters which is generated considering letter frequency in English.

Neural Style Transfer (2018): Implemented the Neural Style Transfer technique to generate artistic images using Keras and TensorFlow, and presented it at the TSEC Entrepreneurship Project Expo 2018. It won the team a Runner-up spot.
